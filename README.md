# Weathering With You

An AI-powered hyperlocal weather app for Pakistani cities, built with FastAPI + React + PyTorch.

**Live endpoints:**
- `/` — React frontend (Gen-Z weather chat with Zero)
- `/mlops` — MLOps dashboard
- `/api/ml/forecast?city=Lahore` — Live Transformer forecast
- `/api/health` — System health
- `/docs` — FastAPI auto-docs

**Models:** Weather Transformer (champion, val_MSE=0.2543), BiLSTM, CNN, β-VAE anomaly detector, FAISS RAG, CLIP
